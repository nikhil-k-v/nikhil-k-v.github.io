LIQUID PISTON PNEUMATIC ENGINE

My final projects involving cycloids and parametric functions for now, I made a liquid piston engine using the same Desmos cycloid generator I created for the robotic arm project! All I needed to do was play around with the parameters until I found something which would work smoothly

The actual numbers I used and plugged into Fusion's equation driven curve add-in are below in the Desmos API embed.

This is one of the nicest looking projects I have ever made, and I'm super lucky that I got to use the laser cutters at my school to make all these acrylic parts, because it would be nothing if it weren't see-through. If you want to make something similar for yourself, I used layers of 1/8" acrylic, both blue and clear, along with MR106 bearings, M3 bolts, and synthetic grease.

There is an intake and exhaust at the top, so it doesn't function in the same ignition cycle as an actual liquid piston engine, but my main goal was to exploit my Equation Driven Curve/Desmos API setup to see if I could actually make the geometry just by modifying parameters. For something more functional made with that, definetely check out the cycloidal actuators I made for an actual robotic arm!
